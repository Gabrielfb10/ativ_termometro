# Atividade Termômetro

Este programa em C apresenta um menu com diferentes funcionalidades:

1. Imprimir números de 1 até N
2. Calcular o fatorial de um número
3. Converter temperaturas (Celsius ↔ Fahrenheit)
4. Contar caracteres de uma string
5. Verificar se um número é primo

O usuário escolhe uma opção no menu e o programa executa a operação correspondente.

---

## Requisitos

Para rodar o programa é necessário ter:

- Um compilador C instalado (ex: **GCC**)
- Terminal ou prompt de comando
---

## Como compilar

No terminal, navegue até a pasta onde está o arquivo `.c` e execute:

```bash
gcc programa.c -o calc

./calc
```
